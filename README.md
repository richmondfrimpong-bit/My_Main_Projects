### Data Analysis on 34-year youth unemployment dataset with Postgres
**Tech :**  PostgreSQL, Tableau

**Description :**
This was a data analysis done on a 34-year youth unemployment dataset acquired from
Kaggle. The analysis unraveled a lot of insights. The most intriguing one being that 
Qatar was the country to have experienced the lowest unemployment rate ever recorded within the 
34-year period in the year 2019, in the face of covid-19. Analysis was done in PostgreSQL and 
visualization covered on Tableau.

**Link :** [-> GitHub Repo] (https://github.com/richmondfrimpong-bit/Postgres-Project)
